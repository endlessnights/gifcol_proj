# gifcol_proj
Library for images, videos & gifs using Django 
(Python)

What is available:

+ Authorization
+ registration
+ User profile
+ Uploading an avatar and cover image to profile
+ Adding additional information about a user
+ Upload photo / video / gif
+ Pre-moderation of uploaded posts
+ Likes (add to favorites / bookmarks)
+ Adding tags / tags to posts
+ Limitations on the size and type of uploaded files
+ Search by post and tag names
+ Used by PostgreSQL
+ Files are uploaded to Amazon S3
