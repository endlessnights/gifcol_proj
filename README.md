# gifcol_proj
Library for memes and gifs
Что имеется:
+ Авторизация
+ Регистрация
+ Профиль пользователя
+ Загрузки аватарки и обложки
+ Добавление дополнительной информации о пользователе
+ Загрузка фото/видео/gif
+ Премодерация загруженных записей
+ Лайки (добавление в избранное/закладки)
+ Добавление тегов/меток к записям
+ Ограничения на размер и тип загружаемых файлов
+ Поиск по названию записей и тегов
+ PostgreSQL
+ Файлы загружаются на Amazon S3
